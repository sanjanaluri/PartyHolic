## Project Description

A web-application that connects people with parties going on around them.

<img width="1552" alt="image" src="https://user-images.githubusercontent.com/15062925/164343096-7a75406b-7c29-4acd-a3cd-2eb2bd2443e3.png">

#### Features

1. **PartyHolic shows all active parties going on around user, using user's Geoloacation**

  ![image](https://user-images.githubusercontent.com/15062925/164343082-5f7ee271-9d55-4971-9ba9-ac6ea8597335.png)

2. **PartyHolic lets user create an account and host/ create parties**

  ![image](https://user-images.githubusercontent.com/15062925/164343052-64f8cf1d-e8fb-4fe1-b0d1-307e3d5b6cc3.png)

3. **PartyHolic lets user see detailed information of the party along with Google Map location**

  ![image](https://user-images.githubusercontent.com/15062925/164343030-b40e2947-152d-47c2-996f-91cec27b490d.png)
4. **PartyHolic shows number of people attending / interested in listed parties**

5. **PartyHolic lets user show interest/ say they attending a party**
